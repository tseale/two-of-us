# Miller Time

Baby tracking web app for Taylor and his wife to track their newborn son Miller.

## Status: Research & design phase — no code yet.

## Key requirements
- Mobile-friendly web app (not native iOS)
- Two users (both parents) tracking the same baby
- Real-time sync so both see updates immediately
- Quick logging — as few taps as possible (you're holding a baby)
- Clean, calming design (not clinical)

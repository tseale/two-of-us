# Miller Time 🍼

Baby tracking web app for newborn care — feeding, sleeping, and diaper changes.

## Status: Research & Design Phase

## Planned Features
- Log feedings (breast/bottle, side, duration, amount)
- Log diaper changes (wet/dirty/both)
- Log sleep (start/stop, duration)
- Timeline view of the day
- Works on mobile for both parents
- Real-time sync between devices
